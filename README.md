Question 2(Inventory Checker py): Script models an inventroy management tool by prompting for current inventory and a reorder threshold. It generates a reorder alert when inventory is low and safely handles invalid input and division-by-zero errors when calculating percentages.

Question 5(Salar Adjustment py): This program simulates employee salary adjustments by validating salary and percentage inputs, enforcing logical constraints (such as preventing negative salaries and percentages outside 0–100), and calculating the updated salary.
